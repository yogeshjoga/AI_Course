Q1. Which of the following is an ultra-fast Python package installer and resolver written in Rust?

A. pip
B. pipenv
C. uv
D. virtualenv

Answer:

**Answer:** C

Explanation:

`uv` is an extremely fast Python package installer and resolver designed as a drop-in replacement for `pip`, written in Rust by Astral.

---

Q2. What is the primary purpose of a Python Virtual Environment?

A. To run Python code in a web browser
B. To isolate project dependencies from the global system packages
C. To compile Python code into machine code
D. To generate databases

Answer:

**Answer:** B

Explanation:

Virtual environments isolate project-specific dependencies, preventing conflicts between different projects on the same machine.

Q3. Which IDE is most widely used and highly recommended for modern Python and FastAPI development due to its rich extension ecosystem?

A. Visual Studio Code (VS Code)
B. Notepad++
C. Microsoft Word
D. Sublime Text 2

Answer:

**Answer:** A

Explanation:

VS Code provides incredible support for Python via official extensions, offering features like Pylance for type checking, Jupyter notebook support, and direct FastAPI debugging.
