Q1. Why are GPUs preferred over CPUs for training deep learning models?

A. CPUs do not support arithmetic operations.
B. GPUs excel at parallel execution of matrix multiplications across thousands of cores.
C. GPUs have larger sequential cache systems than CPUs.
D. GPUs run single-threaded tasks faster.

Answer:

✅ B. GPUs excel at parallel execution of matrix multiplications across thousands of cores.

Explanation:

GPUs are designed for highly parallel matrix computations, which are the core mathematical operations in deep learning.

---

Q2. What is the function of CUDA Cores in NVIDIA GPUs?

A. They compile python code into binary executables.
B. They manage browser API requests.
C. They perform parallel floating-point mathematical computations.
D. They act as central databases for storage.

Answer:

✅ C. They perform parallel floating-point mathematical computations.

Explanation:

CUDA cores are the parallel processing units on NVIDIA GPUs responsible for executing mathematical calculations.

---

Q3. Which processor is purpose-built by Google specifically for machine learning workloads?

A. Central Processing Unit (CPU)
B. Tensor Processing Unit (TPU)
C. Network Processing Unit (NPU)
D. CUDA Hub

Answer:

✅ B. Tensor Processing Unit (TPU)

Explanation:

TPUs (Tensor Processing Units) are custom ASICs developed by Google to accelerate machine learning workloads.
