Q1. What is the role of a DNS (Domain Name System) server?

A. To encrypt raw user passwords.
B. To translate human-readable domain names (like google.com) into machine IP addresses.
C. To route websocket connections.
D. To host database transaction logs.

Answer:

✅ B. To translate human-readable domain names (like google.com) into machine IP addresses.

Explanation:

DNS translates human-friendly domain names to numerical IP addresses to locate systems on the internet.

---

Q2. In a typical Client-Server architecture, what role does the browser play?

A. It executes backend database transactions.
B. It acts as the Client, requesting resources and rendering the UI.
C. It acts as the Server, processing REST methods.
D. It routes messages inside Kafka brokers.

Answer:

✅ B. It acts as the Client, requesting resources and rendering the UI.

Explanation:

The browser is a client that sends requests to the server and displays the returned HTML/CSS/JS content.

Q3. What does API stand for in software development?

A. Automated Program Integration
B. Application Programming Interface
C. Algorithmic Process Index
D. Asynchronous Packet Interceptor

Answer:

✅ B. Application Programming Interface

Explanation:

API stands for Application Programming Interface, which allows different software components to communicate.
