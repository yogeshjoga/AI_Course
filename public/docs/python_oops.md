Q1. What is a Class in Python?

A. A built-in function
B. A blueprint for creating objects
C. A database table
D. A loop structure

Answer:

**Answer:** B

Explanation:

A class provides a means of bundling data and functionality together, serving as a blueprint from which objects (instances) are created.

---

Q2. Which principle of OOP allows a new class to inherit attributes and methods from an existing class?

A. Encapsulation
B. Abstraction
C. Polymorphism
D. Inheritance

Answer:

**Answer:** D

Explanation:

Inheritance enables code reusability by allowing child classes to inherit characteristics from parent classes.

---

Q3. What is the standard name for a constructor method in a Python class?

A. `__init__`
B. `constructor()`
C. `__main__`
D. `create()`

Answer:

**Answer:** A

Explanation:

The `__init__` method is automatically called when a new instance of a class is created.
