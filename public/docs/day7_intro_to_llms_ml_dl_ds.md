Q1. What is the main structural component of Deep Learning (DL) that distinguishes it from classical Machine Learning (ML)?

A. Relational database tables
B. Artificial Neural Networks with multiple hidden layers (Deep Neural Networks)
C. Simple linear decision boundaries
D. Hardcoded rule sheets

Answer:

✅ B. Artificial Neural Networks with multiple hidden layers (Deep Neural Networks)

Explanation:

Deep Learning is a subset of Machine Learning based on Artificial Neural Networks with representation learning. The "deep" refers to the multiple hidden layers inside the network, enabling automatic extraction of high-level features.

Q2. In Data Science, what is the purpose of exploratory data analysis (EDA)?

A. To write the production API
B. To explore, clean, and visualize dataset patterns before building models
C. To host the database in cloud
D. To encrypt user credentials

Answer:

✅ B. To explore, clean, and visualize dataset patterns before building models

Explanation:

EDA is a critical step in Data Science to check data distributions, detect anomalies, identify missing values, and find correlations before applying ML algorithms.

Q3. What represents the 'Transformer' in LLM architectures?

A. A network routing hardware
B. An attention-based neural network architecture that processes sequence inputs in parallel
C. A database indexing tool
D. A python script compiler

Answer:

✅ B. An attention-based neural network architecture that processes sequence inputs in parallel

Explanation:

The Transformer model (introduced by Google in 2017) uses self-attention mechanisms to process tokens concurrently, replacing older sequential architectures like RNNs/LSTMs and forming the base of modern LLMs.
