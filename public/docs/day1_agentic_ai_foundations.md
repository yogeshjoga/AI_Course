Q2. Which statement is most accurate?

A. AI will replace all humans
B. Humans will replace AI
C. People using AI will replace people not using AI
D. AI will stop software development

Answer:

✅ C. People using AI will replace people not using AI

Explanation:

AI acts as a productivity multiplier. Individuals who effectively leverage AI can perform work faster and more efficiently.

Q3. During the Industrial Revolution, machines mainly:

A. Removed farming completely
B. Reduced productivity
C. Multiplied human productivity
D. Replaced all workers

Answer:

✅ C. Multiplied human productivity

Explanation:

Machines helped farmers cultivate significantly more land, increasing output rather than eliminating farming.

Q4. What is Artificial Intelligence (AI)?

B. Software that can make decisions based on data
C. A database system
D. A programming language

Answer:

✅ B. Software that can make decisions based on data

Explanation:

AI systems analyze data and make predictions or decisions, unlike traditional software that follows fixed rules.

Q6. Why is a calculator generally NOT considered AI?

A. It uses electricity
B. It follows fixed rules and does not learn
C. It is too small
D. It has no screen

Answer:

✅ B. It follows fixed rules and does not learn

Explanation:

Calculators execute predefined mathematical operations and do not learn from data or experience.

Q7. What is the correct formula for traditional software?

A. Input + Output = Rules
B. Input + Rules = Output
C. Data + AI = Output
D. Input + AI = Software

Answer:

✅ B. Input + Rules = Output

Explanation:

Developers explicitly define the rules, and software follows those rules to generate outputs.

Q8. What is the correct formula for Machine Learning?

A. Input + Output = Rules
B. Input + Rules = Output
C. Rules + Output = Input
D. AI + Data = Hardware

Answer:

✅ A. Input + Output = Rules

Explanation:

Machine learning algorithms learn the underlying rules by analyzing input and expected output examples.
